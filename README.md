# cleaning-Nike-dataset-and-dashBoard


## 📌 Project Overview
This project focuses on analyzing Nike sales data to uncover valuable business insights and support data-driven decision-making. The analysis includes data cleaning, preprocessing, feature engineering, exploratory data analysis (EDA), and an interactive dashboard for visualizing key performance indicators.

The objective is to transform raw sales data into meaningful information that helps understand sales performance, customer behavior, product trends, and regional performance.


## 🚀 Features
- Data Cleaning & Preprocessing
- Handling Missing Values
- Outlier Detection & Treatment
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Sales & Profit Analysis
- Regional Performance Analysis
- Product Performance Analysis
- Interactive Dashboard
- Business Insights & Recommendations

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Plotly
- Dash
